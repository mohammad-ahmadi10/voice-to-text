the props that the selectBox can accept::

width: "50px"                       set the width of the box 

backgroundColor: "green"            set the background color of the box

height: "50px"                      set the height of the box

fontSize: "18px"

color: "white"

hoverBackgroundColor: "whitesmoke"

hoverColor: "black"

getValue:                           a useCallback function that returns a array of the selected 
                                    Value ["it1" , "item1"]

itemsList:                           a multi dimensional array of items the pattern =>  
                                    [["it1" , "item1"], ["it2" , "item2"], ...]
